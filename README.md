Hieronder staat een stappenplan om dit project te clonen.
1.  Download het ZIP file
2.  Pak het bestand uit
3.  Hernoem het .env.example naar .env
4.  open de terminal en voer dit command uit "composer install"
5.  vervolgens voer je dit command uit "npm install"
6.  daarna "php artisan key:generate" uitvoeren
7.  en dan "php artisan migrate" met daarna "php artisan db:seed" uitvoeren
8.  als laatst "npm run dev" uitvoeren en kan je het project openen
